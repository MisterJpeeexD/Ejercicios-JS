Repositorio especializado en la resolución de problemas lógicos, control de flujo y optimización de código. Funciona como el entorno de pruebas prácticas para aplicar los fundamentos del lenguaje en escenarios reales.

Última actualización: 19-05-2026

---

## 🛠️ Detalle Técnico de los Ejercicios

| Desafío | Enfoque e Instrucciones Implementadas | Objetivos Lógicos |
| :--- | :--- | :--- |
| **Ejercicio 1** | `console.log()` / Control de flujo condicional (`if`). | Salida de datos en consola y validación de igualdades lógicas. |
| **Ejercicio 2** | Interacción dinámica mediante `prompt` / Template Literals (`` `${var}` ``). | Captura de datos en tiempo de ejecución e interpolación limpia de variables. |
| **Ejercicio 3** | Bucles avanzados (`for` clásico, `for...in`, `for...of`, `while(true)`) e instrucciones condicionales (`if`). | Iteración eficiente sobre estructuras de datos (arrays/objetos) y control de bucles infinitos. |
| **Ejercicio 4** | Integración general y refactorización. | Consolidación y optimización de la lógica aplicada en los módulos 1, 2 y 3. |
